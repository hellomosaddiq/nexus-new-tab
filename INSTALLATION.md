# 📦 NEXUS New Tab - Installation Guide

## 🚀 Quick Installation

### Step 1: Download & Extract
1. **Download** `NEXUS-Chrome-Extension-v1.0.0.zip`
2. **Extract** the zip file to a folder (e.g., `NEXUS-Extension/`)
3. **Remember the folder location** - you'll need it for installation

### Step 2: Enable Developer Mode
1. **Open Chrome** and go to `chrome://extensions/`
2. **Toggle "Developer mode"** in the top-right corner
3. You should see new buttons appear: "Load unpacked", "Pack extension", etc.

### Step 3: Load the Extension
1. **Click "Load unpacked"** button
2. **Navigate to** the extracted NEXUS folder
3. **Select the folder** containing `manifest.json`
4. **Click "Select Folder"**

### Step 4: Verify Installation
1. **Look for NEXUS** in your extensions list
2. **Ensure it's enabled** (toggle should be blue/on)
3. **Open a new tab** - you should see NEXUS!

## 🔧 Troubleshooting

### Extension Not Loading?
- **Check folder structure** - `manifest.json` should be in root
- **Refresh extensions page** (`chrome://extensions/`)
- **Try disabling/enabling** the extension

### New Tab Not Showing NEXUS?
- **Check if extension is enabled**
- **Look for error messages** in extensions page
- **Try opening incognito tab** (if allowed)

### Settings Not Saving?
- **Check browser permissions**
- **Try refreshing the page**
- **Check browser console** for errors (F12)
