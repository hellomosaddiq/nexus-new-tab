## 🚀 **Quick Installation (2 Minutes)**

### **Step 1: Download the Extension**
1. Go to [**Releases**](https://github.com/hellomosaddiq/nexus-new-tab/releases/latest)
2. Download `NEXUS-New-Tab-v1.0.0-ChromeWebStore.zip`
3. **Extract the zip file** to a folder on your computer

### **Step 2: Enable Developer Mode**
1. Open **Chrome** browser
2. Go to `chrome://extensions/` (copy-paste this in address bar)
3. Toggle **"Developer mode"** ON (top-right corner)

### **Step 3: Install NEXUS New Tab**
1. Click **"Load unpacked"** button
2. Select the **extracted folder** from Step 1
3. **Done!** NEXUS New Tab is now installed

### **Step 4: Set as New Tab (Important!)**
1. Open a **new tab** - you should see NEXUS New Tab
2. If not, check Chrome settings → Extensions → NEXUS New Tab → Details
3. Ensure **"Allow in incognito"** is enabled if you use private browsing

## 🔧 **Troubleshooting**

### **❓ NEXUS New Tab doesn't appear on new tabs?**
1. Check `chrome://extensions/` - ensure NEXUS New Tab is **enabled**
2. Try **refreshing** the extension (toggle off/on)
3. **Restart Chrome** completely

### **❓ "Load unpacked" button is grayed out?**
1. Make sure **Developer mode** is toggled ON
2. You must be on `chrome://extensions/` page
3. Try **refreshing** the extensions page

### **❓ Extension shows errors?**
1. Make sure you **extracted the zip file** completely
2. Select the **main folder** (not a subfolder)
3. The folder should contain `manifest.json` file

### **❓ Features not working properly?**
1. **Clear browser cache** and restart Chrome
2. Check if any other new tab extensions are **conflicting**
3. Disable other new tab extensions temporarily
