# Keyboard Shortcuts

NEXUS is designed for power users who love keyboard shortcuts. Here's a complete reference of all available shortcuts.

## 🌐 Global Shortcuts

These shortcuts work anywhere in NEXUS:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `K` | Open Quick Shortcuts | Opens the AI-powered shortcuts panel |
| `S` | Open Settings | Opens the tabbed settings panel |
| `N` | Toggle Quick Notes | Shows/hides the notes overlay |
| `Q` | Refresh Daily Quote | Get a new random inspirational quote |
| `F` | Cycle Clock Fonts | Switch between premium typography themes |
| `R` | Reset All Settings | Restore default configuration |
| `Esc` | Close Panel | Closes any open panel or overlay |

## 📝 Quick Notes Shortcuts

When the Quick Notes overlay is open:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+A` | Select All | Selects all text in the notes area |
| `Ctrl+Z` | Undo | Undo last text change |
| `Ctrl+Y` | Redo | Redo last undone change |
| `Esc` | Close Notes | Closes the notes overlay |

## ⚡ Quick Shortcuts Panel

When the Quick Shortcuts panel is open:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Type to search` | Filter Items | Real-time fuzzy search |
| `Enter` | Open Item | Opens the selected item |
| `Tab` | Switch Tabs | Navigate between Bookmarks/Sites/Recent |
| `Esc` | Close Panel | Closes the shortcuts panel |

## ⚙️ Settings Panel

When the Settings panel is open:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Navigate | Move between settings options |
| `Space` | Toggle | Toggle checkboxes and switches |
| `Enter` | Activate | Activate buttons and dropdowns |
| `Arrow Keys` | Navigate | Navigate within dropdown menus |
| `Esc` | Close Settings | Closes the settings panel |

## ⏱️ Focus Timer

The Focus Timer uses mouse controls only:

- **Click Start** - Begin 25-minute timer
- **Click Pause** - Pause current session
- **Click Reset** - Reset timer to 25:00
- Timer can be toggled on/off in Settings

## ♿ Accessibility Shortcuts

Standard browser accessibility shortcuts work throughout NEXUS:

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Tab` | Next Element | Move to next focusable element |
| `Shift+Tab` | Previous Element | Move to previous focusable element |
| `Enter` | Activate | Activate buttons and links |
| `Space` | Toggle | Toggle checkboxes and switches |
| `Arrow Keys` | Navigate | Navigate within menus and lists |

## 🆘 Troubleshooting

### Shortcuts Not Working
1. **Check focus** - Ensure NEXUS has focus (click on the page)
2. **Browser conflicts** - Some shortcuts may be overridden by browser
3. **Extension conflicts** - Other extensions may intercept shortcuts
4. **Refresh page** - Reload the new tab if shortcuts stop working

### Common Conflicts
- **`Ctrl+K`** - May conflict with browser address bar
- **`Ctrl+N`** - Browser new window (use just `N` in NEXUS)
- **`F11`** - Browser fullscreen mode

### Solutions
- **Use NEXUS shortcuts** - Single letters work best (`K`, `S`, `N`)
- **Disable conflicting extensions** - Temporarily disable to test
- **Check browser settings** - Review browser shortcut preferences
