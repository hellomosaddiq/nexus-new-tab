Design Philosophy

NEXUS keyboard shortcuts are designed with these principles:

  • Single Keys - No complex combinations to remember
  • Mnemonic - Letters relate to their function (T for Todo, S for Settings)
  • Non-Conflicting - Avoids browser default shortcuts
  • Accessible - Works with screen readers and assistive technology
  • Consistent - Same shortcuts work across all features

Tips for Mastery

Building Muscle Memory

  • Start Small - Master K, S, and Esc first
  • Daily Practice - Use shortcuts instead of clicking
  • Focus Context - Ensure new tab has focus before pressing keys
  • Gradual Addition - Add one new shortcut per week

Common Workflow Patterns

  • Morning Planning - T → add tasks → Esc → Tab to Start → Space (timer)
  • Quick Navigation - K → type → Enter
  • Customization - S → navigate tabs → make changes → Esc
  • Motivation Break - Q for new quote inspiration
  • Quick Notes - N → jot down thoughts → Esc
  • Theme Switching - F to cycle through typography themes
  • Focus Session - Tab to Start → Space → work → Tab to Pause → Space

Accessibility Features

NEXUS shortcuts are designed for accessibility:

  • Screen Reader Support - All shortcuts announced properly
  • Focus Management - Clear focus indicators and logical tab order
  • Keyboard Only - Complete functionality without mouse
  • High Contrast - Shortcuts work in high contrast mode

Troubleshooting

Shortcuts Not Working

  • Check Focus - Click on the new tab page first
  • Close Other Modals - Press Esc to close any open panels
  • Avoid Input Fields - Shortcuts don't work when typing in text fields
  • Browser Extensions - Other extensions might intercept shortcuts

Conflicting Extensions

If shortcuts don't work, check for conflicts with other extensions that might use the same keys.

Focus Timer Control

The focus timer uses Tab navigation + Space activation:

Starting a Focus Session

  1. Press Tab to navigate to the Start button
  2. Press Space to start the 25-minute timer

Pausing During Session

  1. Press Tab to navigate to the Pause button
  2. Press Space to pause the timer

Resuming After Pause

  1. Press Shift+Tab to go back to Start button
  2. Press Space to resume the timer

Advanced Usage

Power users can combine shortcuts for efficient workflows:

  • Quick Task Entry - T → type task → Enter → repeat
  • Settings Navigation - S → Tab through options
  • Focus Session - Tab to Start → Space → work → Tab to Pause → Space
  • Theme Customization - F to cycle themes → S for fine-tuning
  • Note-Taking Flow - N → capture thoughts → T → convert to tasks
  • Quick Reset - R to restore defaults when experimenting

Troubleshooting

Common keyboard shortcut issues and solutions:

Space Key Behavior

Issue: Pressing Space refreshes quotes instead of controlling focus timer

Solution: The Space key activates whatever button currently has focus. To control the focus timer:

  1. Use Tab to navigate to the specific timer button (Start/Pause)
  2. Then press Space to activate that button
  3. If Space is refreshing quotes, it means the quote refresh button has focus

Focus Management

If shortcuts aren't working as expected:

  • Click somewhere on the page background to reset focus
  • Use Tab to navigate to the desired element first
  • Remember that Space and Enter activate focused buttons
