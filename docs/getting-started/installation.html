Installation Guide

Get NEXUS New Tab Extension up and running

Getting Started • 5 min read • Updated June 25, 2025

Quick Installation

NEXUS is available on the Chrome Web Store for easy one-click installation. For developers, manual installation options are also available.

Chrome Web Store Installation

The easiest way to install NEXUS is through the Chrome Web Store:

  1. Visit the Chrome Web Store

    Go to the Chrome Web Store and search for "NEXUS New Tab" or visit our direct listing page.

  2. Add to Chrome

    Click the "Add to Chrome" button on the NEXUS extension page.

  3. Confirm Installation

    Click "Add extension" in the confirmation dialog that appears.

  4. Start Using NEXUS

    Open a new tab to see NEXUS in action. The extension will automatically replace your default new tab page.

Manual Installation (Developer Mode)

For developers or users who want to install from source:

Prerequisites

  • Google Chrome, Microsoft Edge, or Chromium-based browser
  • Downloaded NEXUS extension files (ZIP or source code)
  • Basic familiarity with browser developer tools

Installation Steps

  1. Download Extension Files

    Download the latest release from our GitHub Releases page or clone the repository.

  2. Extract Files

    If you downloaded a ZIP file, extract it to a folder on your computer. Remember the location as you'll need it in the next steps.

  3. Open Chrome Extensions

    Navigate to chrome://extensions/ in your browser's address bar.

  4. Enable Developer Mode

    Toggle the "Developer mode" switch in the top-right corner of the extensions page.

  5. Load Unpacked Extension

    Click "Load unpacked" and select the folder containing the extracted NEXUS files.

  6. Verify Installation

    NEXUS should now appear in your extensions list. Open a new tab to confirm it's working.

Post-Installation Setup

After installation, you can customize NEXUS to your preferences:

  • Access Settings: Press S or click the gear icon
  • Choose Theme: Select from 9 beautiful color themes
  • Select Typography: Pick from 5 premium font combinations
  • Configure Features: Enable or disable specific features
  • Set Preferences: Customize time format, date insights, and more

Troubleshooting

Extension Not Loading

  • Ensure you're using a compatible browser version
  • Check that Developer mode is enabled (for manual installation)
  • Verify all extension files are present in the folder
  • Try refreshing the extensions page and reloading the extension

New Tab Not Changing

  • Close all existing tabs and open a new one
  • Check if another new tab extension is installed and conflicting
  • Disable other new tab extensions temporarily
  • Restart your browser after installation

Features Not Working

  • Ensure the extension has necessary permissions
  • Check browser console for any error messages
  • Try disabling and re-enabling the extension
  • Clear browser cache and cookies if issues persist
