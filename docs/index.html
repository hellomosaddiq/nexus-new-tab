Powerful Features for Modern Productivity

Every feature is designed with performance, accessibility, and user experience in mind.

AI-Powered Quick Shortcuts

Machine learning algorithms learn your browsing patterns and predict your most-used tabs. Access bookmarks, top sites, and recent tabs with intelligent fuzzy search.

  • Smart tab prediction with 0.42 confidence threshold
  • Real-time fuzzy search across all data sources
  • 45MB IndexedDB caching for instant performance

12 Smart Date Insights

Choose up to 3 contextual date features that provide meaningful information about time, progress, and productivity. From week numbers to moon phases.

  • Year progress, quarter tracking, weekend countdown
  • Moon phases, seasonal info, working days remaining
  • Customizable selection with minimum validation

9 Color + 5 Typography Themes

Professional color themes and premium font combinations. All fonts are loaded locally for instant rendering and zero external dependencies.

  • Blue, Purple, Green, Orange, Pink, Red, Cyan, Yellow, Indigo
  • Inter, Geist, Satoshi, SF Pro, Segoe UI combinations
  • Dynamic theming with CSS custom properties

Pomodoro Focus Timer

Built-in 25-minute Pomodoro timer with visual progress tracking and smart notifications. Boost your focus with proven productivity techniques.

  • Animated circular progress display
  • Pause/resume with state persistence
  • Keyboard shortcuts for quick control

Quick Notes

Instant note-taking with auto-save functionality. Capture thoughts, meeting notes, and ideas without leaving your new tab.

  • 1-second debounced auto-save
  • Dual storage strategy for reliability
  • Full keyboard support with shortcuts

WCAG 2.1 AA Accessible

Built with accessibility as a core principle. Full screen reader support, keyboard navigation, and high contrast compliance.

  • Complete ARIA implementation
  • 4.5:1 minimum contrast ratios
  • Full keyboard navigation support
