Quick Shortcuts

AI-powered command palette for instant navigation and productivity

Features • 6 min read • Updated June 25, 2025

AI-Powered Command Palette

NEXUS features an advanced command palette system with AI-powered tab prediction, universal search engines, and intelligent caching. Designed for all users with privacy, learning, knowledge, shopping, and eco-friendly search options.

Overview

The Quick Shortcuts feature is NEXUS's most powerful productivity tool - an AI-powered command palette that provides instant access to your bookmarks, top sites, recent tabs, and universal search engines. With machine learning-based predictions and local logo assets, it adapts to your browsing patterns while serving all types of users.

Core Features

  • AI-Powered Predictions - Machine learning algorithms predict your most likely tabs
  • Instant Access - Press K for immediate command palette access
  • Universal Search Engines - 5 carefully selected engines for all user types
  • Local Logo Assets - Fast-loading SVG logos stored locally
  • Web Search - Instant search across 5 search engines
  • Cross-Browser Compatibility - Works seamlessly across Chrome, Firefox, and Edge
  • Keyboard Navigation - Complete functionality without mouse interaction

Getting Started

Quick Shortcuts panel (K key) showing bookmarks, recent tabs, and universal search engines

Opening Quick Shortcuts

Access the command palette using the keyboard shortcut:

  • Primary Method - Press K when the new tab has focus
  • Alternative - Click the search icon in the interface (if visible)
  • Context-Aware - Only works when not typing in input fields
  • Instant Response - Opens immediately with AI-predicted tab selected

First Use Experience

  1. Open Command Palette

    Press K to open the Quick Shortcuts panel

  2. Explore Tab Types

    Notice the three tabs: Bookmarks, Top Sites, and Recent Tabs

  3. Use Search Engines

    Type your query and press Enter for Google, or click any search engine below

  4. Use Keyboard Navigation

    Use arrow keys to navigate and Enter to select items

Tab Types

Bookmarks Tab

Access your browser bookmarks with intelligent organization:

  • Recursive Scanning - Searches through all bookmark folders
  • Favicon Caching - Fast-loading icons with IndexedDB persistence
  • Fallback Content - Popular sites when no bookmarks exist
  • Consistent Layout - Always shows exactly 6 items

Top Sites Tab

Your most frequently visited websites with analytics:

  • Browser Analytics - Uses browser's built-in top sites data
  • Visit Frequency - Ordered by actual usage patterns
  • Cross-Browser Support - Works with Chrome, Firefox, and Edge APIs
  • Intelligent Filtering - Excludes internal browser pages

Recent Tabs Tab

Recently closed tabs with AI-powered ranking:

  • Session Restoration - Attempts to restore closed tabs when possible
  • History Integration - Fills gaps with recent browser history
  • AI Ranking - Machine learning-based relevance scoring
  • Deduplication - Removes duplicate URLs for clean interface

Data Loading Strategy

Quick Shortcuts uses concurrent loading for optimal performance:

  1. Parallel Loading

    All data sources (bookmarks, top sites, recent tabs) load simultaneously

  2. Error Isolation

    Individual loader failures don't affect other data sources

  3. Fallback Content

    Popular sites used when browser data is unavailable

  4. Consistent UI

    Always maintains 6-item layout regardless of available data

Troubleshooting

Quick Shortcuts Not Opening

  • Ensure the new tab page has focus when pressing K
  • Check that no input fields are focused
  • Verify NEXUS extension is enabled and up to date
  • Try refreshing the page if the feature seems unresponsive

Missing Bookmarks or Sites

  • Check browser permissions for the NEXUS extension
  • Ensure bookmarks exist in your browser
  • Verify top sites have been visited recently
  • Try opening and closing some tabs to populate recent tabs

Slow Loading or Missing Favicons

  • Check internet connection for favicon downloading
  • Clear browser cache if favicons appear corrupted
  • Disable other extensions that might interfere with image loading
  • Wait for the caching system to build up over time

Privacy & Data

  • Local Processing - All data processing happens on your device
  • No External Transmission - Browsing data never leaves your computer
  • Browser APIs Only - Uses standard browser APIs for data access
  • Favicon Service - Only uses Google's favicon service for icon loading
