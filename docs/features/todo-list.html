Todo List

Built-in task management with persistent storage and keyboard shortcuts

Features • 7 min read • Updated June 25, 2025

Instant Task Management

NEXUS includes a built-in todo list that transforms your new tab into a productivity hub. Access your tasks instantly with the T shortcut and keep everything organized across browser sessions.

Overview

The Todo List feature provides a clean, distraction-free task management system directly in your new tab. Tasks are automatically saved and persist across browser sessions, making it perfect for quick task capture and daily planning.

Key Features

  • Instant Access - Press T to open from anywhere
  • Persistent Storage - Tasks automatically saved across browser sessions
  • Clean Interface - Distraction-free design focused on productivity
  • Keyboard Navigation - Full keyboard support for efficiency
  • Task Counter - Visual progress indicator with completion tracking
  • Drag & Drop - Reorder tasks by dragging them

Getting Started

Quick Notes overlay demonstrating note-taking capabilities that complement the todo list system

Enabling the Todo List

  1. Open Settings

    Press S or click the gear icon to open the settings panel

  2. Navigate to Features

    Click on the "Features" tab in the settings panel

  3. Enable Todo List

    Toggle the "Todo List" switch to enable the feature

  4. Close Settings

    Press Escape or click outside the panel to close settings

Adding Your First Task

  1. Open Todo List

    Press T to open the todo panel

  2. Type Your Task

    The input field is automatically focused - start typing immediately

  3. Add Task

    Press Enter to add the task to your list

  4. Continue Adding

    The input stays focused for quick entry of multiple tasks

Task Management

Adding Tasks

There are several ways to add tasks efficiently:

  • Quick Entry - Press T, type task, press Enter
  • Batch Entry - Add multiple tasks quickly as input stays focused
  • Auto-Save - Tasks are saved immediately when added
  • No Length Limit - Tasks can be as detailed as needed

Completing Tasks

Mark tasks as complete using these methods:

  • Click Checkbox - Click the circle next to any task
  • Keyboard - Focus a task and press Space
  • Visual Feedback - Completed tasks show with checkmark and strikethrough
  • Toggle State - Click again to mark as incomplete

Deleting Tasks

Remove tasks when they're no longer needed:

  • Delete Button - Click the × button next to any task
  • Immediate Action - No confirmation dialog for quick workflow
  • Permanent Removal - Deleted tasks are removed from storage

Reordering Tasks

Organize tasks by priority using drag and drop:

  • Drag Handle - Click and drag any task to reorder
  • Visual Feedback - Clear indicators during drag operation
  • Auto-Save - New order is saved automatically
  • Keyboard Alternative - Use arrow keys when task is focused

Task Counter

The task counter provides real-time progress tracking:

Counter States

  • No Tasks - Shows "0 tasks" when list is empty
  • In Progress - Shows "2/5 completed" format for active lists
  • All Complete - Shows "All 5 completed" when everything is done
  • Live Updates - Counter updates immediately when tasks change

Visual Indicators

  • Header Display - Counter appears in the todo panel header
  • Progress Tracking - Helps visualize completion progress
  • Motivation - Provides positive feedback for completed tasks

Data Storage

Automatic Saving

Tasks are saved automatically using a robust storage system:

  • Instant Persistence - Tasks saved immediately when added or modified
  • Cross-Session - Tasks persist when you close and reopen browser
  • Reliable Storage - Uses browser extension storage for maximum reliability
  • Backup System - Falls back to localStorage if extension storage fails

Data Structure

Each task contains the following information:

  • Unique ID - Timestamp-based identifier for tracking
  • Task Text - The actual task description
  • Completion Status - Boolean flag for completed state
  • Creation Date - ISO timestamp for when task was created

Storage Locations

  • Primary - Browser extension storage (chrome.storage.local)
  • Backup - Browser localStorage as fallback
  • Privacy - All data stored locally on your device
  • Sync - No cloud sync - data stays on your computer

Troubleshooting

Todo Panel Not Opening

  • Ensure the Todo List feature is enabled in settings
  • Check that the new tab page has focus when pressing T
  • Verify no input fields are focused when using the shortcut
  • Try clicking the settings gear and manually enabling the feature

Tasks Not Saving

  • Check browser permissions for the NEXUS extension
  • Ensure you're not in incognito/private browsing mode
  • Try refreshing the page and re-adding tasks
  • Check browser storage settings and available space

Keyboard Shortcuts Not Working

  • Click on the new tab page to ensure it has focus
  • Close any open panels with Escape first
  • Avoid using shortcuts while typing in input fields
  • Check for conflicts with other browser extensions
