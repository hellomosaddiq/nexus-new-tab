Power User Configuration

NEXUS provides comprehensive advanced settings for power users, including feature toggles, data management, system controls, and debugging tools. Configure every aspect of your NEXUS experience with precision.

Overview

Advanced Settings in NEXUS provide granular control over every aspect of the extension. From feature toggles and visual effects to data management and AI system controls, these settings allow power users to customize their experience precisely to their needs.

Key Features

  • Comprehensive Feature Toggles - Enable/disable any NEXUS feature
  • Visual Effect Controls - Fine-tune animations and visual elements
  • Data Management - Reset, backup, and manage all user data
  • AI System Controls - Manage machine learning and tab prediction
  • Debug Tools - Advanced debugging and analytics access
  • Storage Management - Monitor and control data storage

    Use the tab navigation to access different setting categories

    Most settings apply immediately without requiring confirmation

Storage and Performance

Storage Strategy

NEXUS uses a sophisticated multi-tier storage system:

Primary Storage (Sync)

  • Chrome Sync Storage - 8KB limit, cross-device synchronization
  • Settings Sync - Themes, preferences, and configurations
  • Automatic Backup - Redundant storage for reliability
  • Quota Management - Intelligent space usage

Local Storage (Fallback)

  • Extension Storage - 10MB limit, device-specific
  • AI Learning Data - Tab memory and usage patterns
  • Performance Cache - Favicon cache and temporary data
  • Offline Functionality - Works without internet

Emergency Fallback

  • localStorage - Browser-native storage
  • Critical Settings - Essential configuration only
  • Graceful Degradation - Maintains functionality
  • Automatic Recovery - Restores when storage available

Debug and Development Tools

Debug Mode

Advanced debugging tools for power users and developers:

Enabling Debug Mode

  1. Open Browser Console

    Press F12 or right-click → Inspect → Console

  2. Enable Debug Mode

    Run: localStorage.setItem('nexus-debug-mode', 'true')

  3. Refresh Page

    Reload the new tab to activate debug features

  4. Access Debug Tools

    Use window.debugNexus object for advanced functions

Available Debug Functions

  • Tab Memory Analytics - debugNexus.getTabMemoryAnalytics()
  • Reset Tab Memory - debugNexus.resetTabMemory()
  • Settings Inspection - debugNexus.getSettings()
  • Performance Metrics - debugNexus.getPerformanceData()

Troubleshooting Advanced Issues

Settings Not Saving

  • Check browser permissions for NEXUS extension
  • Ensure sufficient storage space available
  • Try disabling other extensions temporarily
  • Reset to defaults and reconfigure gradually

Performance Issues

  • Disable animations if experiencing lag
  • Reduce number of enabled features
  • Clear browser cache and restart
  • Check for conflicting extensions

AI System Issues

  • Reset tab memory if predictions seem incorrect
  • Allow 2-3 days for AI to learn new patterns
  • Check that Quick Shortcuts is enabled
  • Verify sufficient usage data exists

Configuration Philosophy

NEXUS advanced settings follow a "progressive disclosure" philosophy - essential features are easily accessible, while power user options are available but not overwhelming. Every setting has a purpose and provides real value to specific use cases.
